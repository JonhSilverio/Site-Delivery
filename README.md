## ‚ú® Key Features

- **Modern & Responsive UI:** Fully mobile-friendly interface, designed to look great on all devices.
- **Dynamic Shopping Cart:** Users can add/remove items, and the total price updates instantly using LocalStorage (data persists even if the page is refreshed).
- **WhatsApp Checkout:** The order details (items, customer name, address, payment method, and total) are formatted and sent directly to a WhatsApp number, making it perfect for small businesses.
- **Live Search & Filtering:** Users can search for burgers by name or description in real-time.
- **Horizontal Scroll Menus:** Netflix-style sliding rows for different categories (Combos, Promos, Best Sellers).
- **Interactive Modals & Menus:** Smooth animations for the mobile navigation bar and contact modals.

## üõ†Ô∏è Technologies Used
